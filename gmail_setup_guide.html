📧 Gmail Setup Guide for Email System

⚠️ Important: You MUST use an App Password, not your regular Gmail password. Regular passwords will not work with SMTP.

Step 1: Enable 2-Factor Authentication

  1. Go to your Google Account Security page
  2. Under "Signing in to Google", click on "2-Step Verification"
  3. Follow the setup process to enable 2FA (you'll need your phone)
  4. Make sure 2-Step Verification shows as "On"

Step 2: Generate App Password

  1. Go to Google App Passwords
  2. You might need to sign in again
  3. In the "Select app" dropdown, choose "Mail"
  4. In the "Select device" dropdown, choose "Other (Custom name)"
  5. Type "Email System" as the custom name
  6. Click "Generate"
  7. Copy the 16-digit password (format: xxxx xxxx xxxx xxxx)
Important: Save this password immediately! Google will only show it once.

Step 3: Update Email Configuration

Open config/email_config.php and update these lines:

define('EMAIL_PROVIDER', 'gmail');
define('GMAIL_USERNAME', 'soumyaranjanpadhi936@gmail.com'); // ← Your Gmail address
define('GMAIL_APP_PASSWORD', 'tpvfzevkdebtuoyt'); // ← Your 16-digit app password (no spaces!)

Example:

define('EMAIL_PROVIDER', 'gmail');
define('GMAIL_USERNAME', 'john.doe@gmail.com');
define('GMAIL_APP_PASSWORD', 'abcdabcdabcdabcd'); // Remove spaces from the app password

Step 4: Test Your Configuration

  1. Save the email_config.php file
  2. Go back to the Send Email page
  3. The warning message should disappear
  4. Send a test email to yourself

✅ Configuration Complete!

Once you've completed these steps, you should be able to send emails successfully.

Go to Send Email

🔧 Troubleshooting

Common Issues:

❌ "SMTP Error: Could not authenticate"

  • Make sure you're using the App Password, not your regular Gmail password
  • Check that 2-Factor Authentication is enabled
  • Verify your Gmail address is correct
  • Make sure there are no spaces in the app password

❌ "App Passwords" option not available

  • 2-Factor Authentication must be enabled first
  • Some Google Workspace accounts may have restrictions
  • Try using a personal Gmail account instead

❌ "Connection timed out"

  • Check your internet connection
  • Verify firewall isn't blocking port 587
  • Try using port 465 with SSL instead

Alternative Email Providers:

If Gmail doesn't work, you can try:

  • Outlook/Hotmail: Change EMAIL_PROVIDER to 'outlook'
  • Yahoo: Change EMAIL_PROVIDER to 'yahoo'
  • Custom SMTP: Use your hosting provider's SMTP settings

🔒 Security Tips:

  • Never share your app password with anyone
  • If you suspect it's compromised, revoke it and generate a new one
  • Use different app passwords for different applications
  • Keep your email configuration file secure
